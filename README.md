<h1>Pomodoctivity Pro</h1>

<p>
Pomodoctivity Pro is a modern productivity application built using
<strong>React</strong>. It helps users stay focused using the Pomodoro
Technique while providing detailed productivity analytics, customizable
timer settings, notifications, and theme personalization.
</p>

Live Link : https://pomoductivity-pro.vercel.app/

<h2>Features</h2>

<ul>

<li>Customizable Pomodoro timer</li>

<li>Separate timer configuration for:
<ul>
<li>Focus Session</li>
<li>Short Break</li>
<li>Long Break</li>
</ul>
</li>

<li>Circular progress indicator</li>

<li>Focus session tracking</li>

<li>Today's total focus duration</li>

<li>Weekly focus duration</li>

<li>Current productivity streak</li>

<li>Interactive focus analytics dashboard</li>

<li>Weekly focus chart</li>

<li>Monthly focus chart</li>

<li>Multiple application themes</li>

<li>Browser notifications</li>

<li>Reminder notifications before timer completion</li>

<li>Option to notify:
<ul>
<li>Last <i>X</i> minutes before completion</li>
<li>Every <i>X</i> minutes</li>
</ul>
</li>

<li>Custom alarm sound selection</li>

<li>Automatic timer completion alerts</li>

<li>Persistent settings using Local Storage</li>

<li>Responsive user interface</li>

</ul>
